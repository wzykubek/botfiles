# Installation
To install setup go to repo on your disk and use `./installer.py` script.

## My setup
* distro: arch linux
* browser: qutebrowser
* terminal: st
* file manager: ranger
* app menu: dmenu
* text editor: neovim
* window manager: i3
* status bar: polybar
* mail client: neomutt
* aur helper: yay
* shell: zsh
* music client: mopidy
* music player: ncmpcpp
